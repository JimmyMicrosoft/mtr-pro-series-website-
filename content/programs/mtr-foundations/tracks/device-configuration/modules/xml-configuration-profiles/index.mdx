## Overview

XML configuration profiles are the advanced configuration layer for MTR Windows. While the Teams app settings interface covers the most common options, SkypeSettings.xml provides access to a broader set of parameters — including display behavior, meeting defaults, audio routing, content camera assignment, and custom branding — in a format that is portable, version-controllable, and deployable at scale across an entire fleet without touching each device individually.

This module covers the SkypeSettings.xml schema, how to write XML for common configuration scenarios, and the deployment methods available — from manual file placement to fleet-wide deployment through Intune and Teams Admin Center.

---

## What you will learn

By completing this module, you will be able to:

- Read and write valid SkypeSettings.xml — understanding the root element, key configuration categories, case sensitivity requirements, and silent-fail behavior
- Build XML configurations for the most common MTR deployment scenarios: dual display, content camera assignment, custom theming, and Front Row defaults
- Deploy XML configuration using manual file copy, USB drive, Intune PowerShell script, and TAC config profiles — and explain the precedence order when multiple methods are in use simultaneously
