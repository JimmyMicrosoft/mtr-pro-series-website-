## Overview

Cable infrastructure is the foundation of an MTR installation. Every device in the system is connected to something else, and every connection is only as reliable as the cable and termination behind it. Poor cable work — wrong cable type, passive runs beyond their limits, undertorqued punchdowns, unmanaged bundles, unlabeled runs — creates problems that are expensive to diagnose and more expensive to fix after the walls are closed.

This module covers the complete cable knowledge set for professional MTR installation: what to use, how far it goes, how to terminate it correctly, how to verify it with test equipment, how to manage and label it, and how to calculate the power requirements that run through it.

---

## What you will learn

By completing this module, you will be able to:

- Select the correct cable type for every MTR signal path — network, display, USB, audio, and control
- State passive distance limits for HDMI, USB 3.x, USB 2.0, and Cat6/6A, and select the appropriate extension technology when those limits are exceeded
- Terminate Cat6/6A to T568B standard at RJ45 plugs and 110-style punchdown blocks using correct technique
- Perform a post-termination cable certification test, interpret the results, and document them in a cable schedule
- Apply ANSI/TIA-606-C-based cable labeling conventions and professional cable management practices
- Calculate PoE power budgets and UPS VA requirements for MTR room systems
